# FullscreenModule

Provides a cross-browser directive to toggle fullscreen 

**Installation**

```Shell
npm install --save screenfull
```

- Import `FullscreenModule` from `@app/fullscreen`

**Usage**

```html
<button pfmFullscreen>Toggle fullscreen<button>
  ```
